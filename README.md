# BUZZ... Street Wear 👟🔥

¡Bienvenido a **BUZZ**, una tienda virtual de ropa urbana diseñada para los amantes del street wear! Este proyecto es el resultado del **Desafío 2**, enfocado en la creación de layouts modernos utilizando técnicas avanzadas de maquetación web.

## 🛠️ Tecnologías Utilizadas

Para este proyecto se utilizaron las siguientes herramientas:

*   **HTML5**: Estructura semántica del sitio.
*   **CSS3**:
    *   **CSS Grid**: Utilizado para el layout principal (`grid-template-areas`) y la grilla de productos.
    *   **Flexbox**: Aplicado para la distribución de la barra lateral (aside) y la alineación interna de las tarjetas de productos.
*   **Bootstrap 5**: Implementación de componentes (botones) y utilidades de espaciado.
*   **Google Fonts / System Fonts**: Tipografía moderna y legible.

---

## 📋 Características del Proyecto

El sitio cumple con los siguientes requerimientos técnicos detallados en la escala de apreciación:

1.  **Layout Principal**: Estructura base construida con **CSS Grid**, definiendo áreas específicas para `nav`, `aside`, `section` y `footer`.
2.  **Grilla de Productos**: Sección de catálogo dinámica creada con una rejilla de CSS Grid de 3 columnas.
3.  **Barra Lateral (Aside)**: Navegación de categorías organizada mediante **Flexbox** para un control total del espaciado.
4.  **Tarjetas de Producto**: Diseño de cards consistente utilizando **Flexbox** para asegurar que el contenido (imagen, título y botón) esté perfectamente alineado.
5.  **Diseño Moderno**: Efectos de *hover* en tarjetas y enlaces para mejorar la experiencia de usuario.
